Messy ARIA (1)

This is a messed up ARIA radiogroup implementation, since:

  • it does not provide the expected keyboard support;
  • the selection state of each radio button is not exposed appropriately to assistive technologies (i.e. using aria-selected instead of aria-selected);
  • the grouping of radio buttons is not exposed properly to assistive technologies (i.e. the container does not use the radiogroup role).
