Tabs examples

Here you can find different tabs examples with various accessibility degrees, implemented both by using native HTML Elements and attributes from the WAI-ARIA specification.

Plain HTML

Or well... HTML does not offer native elements to represent tablists, tabs, and tab panels.

Tabs with manual activation

This is a correct implementation of a tabbed interface that uses attributes from the WAI-ARIA specification and leverages the so-called "manual-activation" technique: basically, this means that moving between tabs using left/right arrow, the home key or the end key will move focus to the appropriate tab, but it won't automatically display the corresponding tab panel.
